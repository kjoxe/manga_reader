# manga_reader

Very minimal raw manga reader for my personal use. Only dependency is PyQt5.

## Basic usage

$ manga_reader zipfile

```
Quit       - q
Next image - j/space
Prev image - k
  (Prepend number to jump more than 1 image at once
  For example pressing "5j" will jump 5 images ahead)
```
